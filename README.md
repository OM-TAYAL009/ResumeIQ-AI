# ResumeIQ-AI

ResumeIQ-AI is a web application that analyzes resumes and provides AI-driven feedback to improve ATS compatibility and job readiness.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- AI API

## Features
- Resume upload and analysis
- Skill and keyword matching
- Instant AI-based feedback

## Getting Started
```bash
npm install
npm run dev
